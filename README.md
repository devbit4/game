# :fish: Project Title: Fishing Game

- 소개: 정해진 시간 안에 상어를 피해 빨간 물고기를 잡는 게임 <br>
        낚시대 초점을 맞추어 물고기를 클릭하면 점수가 누적된다.

- 제작이유

  - 간단한 흥미 위주의 게임 개발

- 사용스킬

  - HTML
  - CSS
  - JavaScript

- 기능

  - function 1) game start
  - function 2) game stop
  - function 3) game re-play
  - function 4) score
  - function 5) timer
